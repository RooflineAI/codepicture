    Function load_config():
    ```python
    def load_config(
        global_path: Path | None = None,
        local_path: Path | None = None,
        cli_overrides: dict[str, Any] | None = None,
        use_defaults: bool = True
    ) -> RenderConfig:
        """Load configuration with precedence: CLI > local > global > defaults.

        Args:
            global_path: Path to global config (default: ~/.config/codepicture/config.toml)
            local_path: Path to local config (default: .codepicture.toml)
            cli_overrides: Dict of CLI flag overrides (None values are filtered out)
            use_defaults: If True, use default paths when global_path/local_path are None

        Returns:
            Validated RenderConfig

        Raises:
            ConfigError: If TOML parsing fails or validation fails
        """
    ```

    Implementation:
    1. If use_defaults and global_path is None, use DEFAULT_GLOBAL_CONFIG_PATH
    2. If use_defaults and local_path is None, use DEFAULT_LOCAL_CONFIG_PATH
    3. Start with empty merged dict
    4. If global_path exists, load and merge (use tomllib, binary mode!)
    5. If local_path exists, load and merge (overwrites global values)
    6. If cli_overrides provided, filter out None values and merge (highest priority)
    7. Call RenderConfig.model_validate(merged)
    8. Catch tomllib.TOMLDecodeError -> raise ConfigError with file/line info
    9. Catch pydantic.ValidationError -> raise ConfigError with field details

    Helper function:
    ```python
    def _load_toml(path: Path) -> dict[str, Any]:
        """Load TOML file, raising ConfigError on failure."""
    ```

    Update src/codepicture/__init__.py to also export RenderConfig and load_config.
