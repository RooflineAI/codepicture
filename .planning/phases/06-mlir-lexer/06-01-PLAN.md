  <action>
Create `src/codepicture/highlight/mlir_lexer.py` with MlirLexer class:

1. Import from pygments.lexer: RegexLexer, bygroups
2. Import token types: Comment, Keyword, Name, Number, Operator, Punctuation, String, Text
3. Create MlirLexer(RegexLexer) with:
   - name = 'MLIR'
   - aliases = ['mlir']
   - filenames = ['*.mlir']
   - mimetypes = ['text/x-mlir']
4. Define tokens dict with 'root' and 'string' states:

Root state patterns (in order):
- Comments: `r'//.*$'` -> Comment.Single
- SSA values: `r'%[\w\.\$\:\#]+'` -> Name.Variable
- Block labels: `r'\^[\w\d_$\.\-]+'` -> Name.Label
- Function refs: `r'@[\w+\$\-\.]+'` -> Name.Function
- Quoted function refs: `r'@"[^"]*"'` -> Name.Function
- Attribute aliases: `r'#[\w\$\-\.]+'` -> Name.Constant
- Type aliases: `r'![\w\$\-\.]+'` -> Keyword.Type
- Built-in types: `r'\b(index|none|bf16|f16|f32|f64|f80|f128)\b'` -> Keyword.Type
- Integer types: `r'\b[su]?i[0-9]+\b'` -> Keyword.Type
- Container types: `r'\b(memref|tensor|vector|complex|tuple)\b'` -> Keyword.Type
- Operations (dialect.op): `r'[\w]+\.[\w\.\$\-]+'` -> Name.Builtin
- Reserved keywords: `r'\b(affine_map|affine_set|dense|opaque|sparse|func|return|module)\b'` -> Keyword.Reserved
- Boolean/unit: `r'\b(true|false|unit)\b'` -> Keyword.Constant
- Affine operators: `r'\b(floordiv|ceildiv|mod|symbol)\b'` -> Operator.Word
- Hex numbers: `r'0x[0-9a-fA-F]+'` -> Number.Hex
- Float numbers: `r'[0-9]+\.[0-9]*([eE][+-]?[0-9]+)?'` -> Number.Float
- Integers: `r'[0-9]+'` -> Number.Integer
- String start: `r'"'` -> String.Double, push 'string' state
- Operators: `r'[+\-*/]'` -> Operator
- Arrow: `r'->'` -> Punctuation
- Punctuation: `r'[()[\]<>,{}=:]'` -> Punctuation
- Whitespace: `r'\s+'` -> Text.Whitespace

String state patterns:
- Escapes: `r'\\[nt"]'` -> String.Escape
- Content: `r'[^"\\]+'` -> String.Double
- End: `r'"'` -> String.Double, pop state

Add module docstring explaining this is a custom MLIR lexer for codepicture.
