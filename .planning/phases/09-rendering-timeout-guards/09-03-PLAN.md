- `test_pipeline_error_propagates`: Patch `generate_image` to raise `RenderError("fail")`, call `generate_image_with_timeout(timeout=5.0)`, assert `RenderError` is raised (not wrapped)

Add `TestAtomicWrite` class:
- `test_atomic_write_creates_file`: Call `_write_output_atomic(b"data", tmp_path / "out.png")`, assert file exists with correct content
- `test_atomic_write_creates_parent_dirs`: Write to `tmp_path / "nested/out.png"`, assert works
- `test_atomic_write_no_partial_on_failure`: Mock shutil.move to raise, assert output file does NOT exist, assert temp file cleaned up
- `test_atomic_write_preserves_existing_on_failure`: Create existing file, then try atomic write that fails during move, assert original file untouched

Import `time` for sleep-based timeout test. Mark timeout tests with `@pytest.mark.timeout(10)` to prevent hangs.
