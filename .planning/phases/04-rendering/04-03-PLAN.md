  <action>
    Create shadow.py with shadow post-processing:

    1. Imports:
       ```python
       import cairo
       from io import BytesIO
       from PIL import Image, ImageFilter
       from codepicture.core.types import Color
       ```

    2. Define shadow constants (per RESEARCH.md, fixed macOS style per CONTEXT.md):
       ```python
       # macOS-style shadow (fixed, not configurable per CONTEXT.md)
       SHADOW_BLUR_RADIUS = 50      # Standard blur for macOS window feel
       SHADOW_OFFSET_X = 0          # Centered horizontally
       SHADOW_OFFSET_Y = 25         # Slight downward offset
       SHADOW_COLOR = Color(0, 0, 0, 128)  # 50% black
       ```

    3. Implement apply_shadow(surface: cairo.ImageSurface, enabled: bool = True) -> bytes:
       - If not enabled: just write_to_png to BytesIO and return

       - Get surface dimensions
       - Calculate shadow_margin = SHADOW_BLUR_RADIUS * 2 + max(abs(SHADOW_OFFSET_X), abs(SHADOW_OFFSET_Y))

       - Convert Cairo surface to Pillow Image:
         * CRITICAL: Cairo uses BGRA (pre-multiplied), use "RGBa" mode in frombytes
         * data = bytes(surface.get_data())
         * pil_image = Image.frombytes("RGBa", (width, height), data)
         * pil_image = pil_image.convert("RGBA")

       - Create shadow from alpha channel:
         * alpha = pil_image.split()[3]
         * shadow = Image.new("RGBA", (width, height), (SHADOW_COLOR.r, SHADOW_COLOR.g, SHADOW_COLOR.b, 0))
         * shadow.putalpha(alpha)
         * shadow = shadow.filter(ImageFilter.GaussianBlur(radius=SHADOW_BLUR_RADIUS))

       - Create output canvas with margin:
         * output_width = width + 2 * shadow_margin
         * output_height = height + 2 * shadow_margin
         * output = Image.new("RGBA", (output_width, output_height), (0, 0, 0, 0))

       - Composite shadow then content:
         * shadow_x = shadow_margin + SHADOW_OFFSET_X
         * shadow_y = shadow_margin + SHADOW_OFFSET_Y
         * output.paste(shadow, (shadow_x, shadow_y), shadow)
         * output.paste(pil_image, (shadow_margin, shadow_margin), pil_image)

       - Export to PNG bytes:
         * buffer = BytesIO()
         * output.save(buffer, format="PNG")
         * return buffer.getvalue()

    4. Add helper function calculate_shadow_margin() -> int:
       - Returns shadow_margin calculation for use by Renderer to pre-expand canvas
  </action>

  </verify>
