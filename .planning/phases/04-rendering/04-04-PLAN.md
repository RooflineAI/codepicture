  <action>
    Create renderer.py with Renderer class:

    1. Imports:
       - From render: CairoCanvas, draw_title_bar, TITLE_BAR_HEIGHT, apply_shadow, calculate_shadow_margin
       - From core.types: Color, OutputFormat, LayoutMetrics, RenderResult
       - From core.protocols: Theme (TYPE_CHECKING)
       - From config.schema: RenderConfig
       - From highlight: TokenInfo (TYPE_CHECKING)

    2. Implement Renderer class:
       ```python
       class Renderer:
           """Orchestrates rendering of syntax-highlighted code to images."""

           def __init__(self, config: RenderConfig) -> None:
               self._config = config

           def render(
               self,
               lines: list[list[TokenInfo]],
               metrics: LayoutMetrics,
               theme: Theme,
           ) -> RenderResult:
               """Render highlighted code to image bytes.

               Args:
                   lines: Tokenized lines from highlighter
                   metrics: Pre-calculated layout dimensions
                   theme: Color theme for styling

               Returns:
                   RenderResult with image data and dimensions
               """
       ```

    3. Render method implementation:
       a) Calculate final dimensions:
          - Base: metrics.canvas_width, metrics.canvas_height
          - Add TITLE_BAR_HEIGHT if window_controls enabled
          - For PNG with shadow: add calculate_shadow_margin() * 2

       b) Create CairoCanvas:
          - scale = 2.0 for PNG (per CONTEXT.md HiDPI)
          - scale = 1.0 for SVG/PDF

       c) Draw background:
          - Full canvas rounded rect with theme.background
          - Use config.corner_radius

       d) Draw window chrome (if window_controls and not WindowStyle.NONE):
          - Call draw_title_bar with theme.background, config.window_title, corner_radius
          - Offset subsequent drawing by TITLE_BAR_HEIGHT

       e) Draw code area:
          - Calculate code_y offset (content_y + title_bar_offset if chrome)
          - If show_line_numbers:
            * For each line: draw line number at gutter_x, baseline_y
            * Use theme.line_number_fg color
          - For each line, for each token:
            * Get style from theme.get_style(token.token_type)
            * Draw text at code_x + column offset, baseline_y
            * Track x position via draw_text return value

       f) Save and apply shadow:
          - For PNG with shadow enabled: apply_shadow(canvas._surface, enabled=True)
          - For PNG without shadow: canvas.save()
          - For SVG/PDF: canvas.save() (no shadow support)

       g) Return RenderResult(data, format, final_width, final_height)

    4. Update __init__.py exports:
       - Export Renderer from render module
       - Add to __all__

    5. Update src/codepicture/__init__.py:
       - Export Renderer from top-level
  </action>
