  <action>
    Create the render module with CairoCanvas implementing Canvas protocol.

    In canvas.py:
    1. Import cairo, math, BytesIO, Path
    2. Import Color, OutputFormat from core.types
    3. Import resolve_font_family from fonts module
    4. Create helper function draw_rounded_rect(ctx, x, y, width, height, radius) using arc paths
    5. Implement CairoCanvas class:
       - __init__(surface, context, format, buffer, scale) storing all as private attributes
       - @classmethod create(width, height, format, scale=1.0) factory method:
         - PNG: ImageSurface at FORMAT_ARGB32 with width*scale, height*scale
         - SVG: SVGSurface with BytesIO buffer at logical dimensions
         - PDF: PDFSurface with BytesIO buffer at logical dimensions
         - Apply context.scale(scale, scale) for PNG
       - @property width, height returning logical dimensions (divide by scale for PNG)
       - draw_rectangle(x, y, width, height, color, corner_radius=0):
         - If corner_radius > 0, use draw_rounded_rect
         - Set source RGBA from Color (divide by 255.0)
         - Call fill()
       - draw_circle(x, y, radius, color):
         - ctx.arc(x, y, radius, 0, 2*math.pi)
         - Set color and fill
       - draw_text(x, y, text, font_family, font_size, color) -> float:
         - resolve_font_family for font name
         - ctx.select_font_face(name, SLANT_NORMAL, WEIGHT_NORMAL)
         - ctx.set_font_size(font_size)
         - Set color, move_to(x, y), show_text(text)
         - Return text_extents(text).x_advance
       - measure_text(text, font_family, font_size) -> tuple[float, float]:
         - Same font setup, return (extents.x_advance, font_extents height)
       - push_clip(x, y, width, height):
         - ctx.save(), ctx.rectangle(), ctx.clip()
       - pop_clip():
         - ctx.restore()
       - save() -> bytes:
         - PNG: BytesIO + write_to_png + getvalue
         - SVG/PDF: surface.finish() then buffer.getvalue()
       - save_to_file(path):
         - path.write_bytes(self.save())

    In __init__.py:
    - Export CairoCanvas from canvas module

    Note: apply_shadow is a no-op stub for now (Plan 04-03 implements real shadow).
  </action>
