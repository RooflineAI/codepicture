  <name>Task 1: Create parametrized visual regression tests and generate initial reference images</name>
  <files>
    tests/visual/test_visual_regression.py
    tests/visual/references/ (directory with generated PNGs)
  </files>
  <action>
    Create `tests/visual/test_visual_regression.py` with:

    1. **Reference naming convention**: `{language}_{format}_default.png`
       Examples: `python_png_default.png`, `rust_svg_default.png`, `mlir_pdf_default.png`
       Note: SVG and PDF references are stored as rasterized PNGs (the rasterization happens at comparison time from fresh render output; the reference is the rasterized version).

    2. **Language/format mapping**:
       ```python
       LANGUAGES = [
           ("python", "python_visual.py"),
           ("rust", "rust_visual.rs"),
           ("cpp", "cpp_visual.cpp"),
           ("javascript", "javascript_visual.js"),
           ("mlir", "mlir_visual.mlir"),
       ]
       FORMATS = ["png", "svg", "pdf"]
       ```

    3. **Test: `test_visual_regression[{language}-{format}]`**
       Parametrize over `LANGUAGES x FORMATS` (15 combinations).

       For each combo:
       a. Build a RenderConfig with `output_format=format` and default settings
       b. Call `render_fixture(fixture_path, config, language=language)` to get raw bytes
       c. Convert to Pillow Image:
          - PNG: `Image.open(BytesIO(data)).convert("RGBA")`
          - SVG: `svg_to_png(data)`
          - PDF: `pdf_to_png(data)`
       d. Construct reference path: `references_dir / f"{language}_{format}_default.png"`
       e. If `snapshot_update` is True OR reference doesn't exist:
          - Save the Pillow Image to the reference path
          - `pytest.skip("Reference image created/updated")`
       f. Otherwise, call `compare_images(actual, reference_path, diff_output_dir, test_name)`
       g. Assert passed, with message including mismatch_pct

    4. **Mark all visual tests with timeout marker**: `@pytest.mark.timeout(30)` — rendering 15 images takes longer than the 5s default.

    5. **Generate initial references**: Run `uv run pytest tests/visual/test_visual_regression.py --snapshot-update -v` to populate the references/ directory.

    6. **Verify references exist**: Assert all 15 reference files exist (5 languages x 3 formats).

    7. **Run tests without --snapshot-update**: `uv run pytest tests/visual/test_visual_regression.py -v` — all 15 should pass.
  </action>
  <verify>
    - `ls tests/visual/references/*.png | wc -l` shows 15 files
    - `uv run pytest tests/visual/test_visual_regression.py -v` — all 15 tests pass
    - Reference filenames follow `{language}_{format}_default.png` convention
  </verify>
