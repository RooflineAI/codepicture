<task type="auto">
  <name>Task 3: Create GitHub Actions CI workflow and verify coverage</name>
  <files>.github/workflows/test.yml</files>
  <action>
1. Create .github/workflows/test.yml:
   ```yaml
   name: Tests

   on:
     push:
       branches: [main]
     pull_request:
       branches: [main]

   jobs:
     test:
       runs-on: ubuntu-latest

       steps:
         - uses: actions/checkout@v4

         - name: Install uv
           uses: astral-sh/setup-uv@v5
           with:
             enable-cache: true

         - name: Set up Python
           run: uv python install 3.13

         - name: Install dependencies
           run: uv sync --locked --dev

         - name: Run tests with coverage
           run: uv run pytest --cov --cov-report=xml --cov-report=term

         - name: Upload coverage to Codecov
           uses: codecov/codecov-action@v4
           with:
             file: coverage.xml
             fail_ci_if_error: false
   ```

2. Run full test suite with coverage:
   ```bash
   uv run pytest --cov --cov-report=term-missing
   ```

3. Verify coverage is at least 80%. If not, identify and add missing tests.
