   - `gutter_indicator_x = 0.0`
   - `gutter_indicator_width = 0.0`
   - Layout unchanged from Phase 12 (per Pitfall 3 -- no layout shift without highlights)

content_width = gutter_width + indicator_width + gap + code_width

b. **Gutter indicators** -- draw after highlight rects, before text, only when line numbers enabled and style_map non-empty. Note: gutter indicators are only drawn on highlighted lines (lines present in style_map). Since highlighted lines are by definition NOT unfocused, no focus dimming applies to gutter indicators:
```python
if config.show_line_numbers and style_map:
    for line_idx in range(len(lines)):
        if line_idx in style_map:
            style = style_map[line_idx]
            indicator_color = resolve_style_color(style, style_overrides)
            # High opacity for visibility (per D-13): use base RGB at ~90% alpha
            indicator_color = Color(
                r=indicator_color.r, g=indicator_color.g, b=indicator_color.b,
                a=max(1, int(indicator_color.a * 255 / 64 * 0.9)) if indicator_color.a < 255 else 230,
            )
            # No focus dimming needed here: gutter indicators only appear on
            # highlighted lines, which are always at full brightness in focus mode.

            baseline_y = (
                metrics.content_y + code_y_offset
                + line_idx * metrics.line_height_px
                + metrics.baseline_offset
            )
            symbol = GUTTER_INDICATORS.get(style)
            if symbol:  # "+" or "-"
                canvas.draw_text(
                    x=metrics.gutter_indicator_x,
                    y=baseline_y,
                    text=symbol,
                    font_family=config.font_family,
                    font_size=config.font_size,
                    color=indicator_color,
                )
            else:  # colored bar for highlight/focus
                bar_y = (
                    metrics.content_y + code_y_offset
                    + line_idx * metrics.line_height_px
                )
                canvas.draw_rectangle(
                    x=metrics.gutter_indicator_x + (metrics.gutter_indicator_width - GUTTER_BAR_WIDTH) / 2,
                    y=bar_y,
                    width=GUTTER_BAR_WIDTH,
                    height=metrics.line_height_px,
                    color=indicator_color,
                )
```
